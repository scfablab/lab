＜１. 概要＞

CutStudio は、当社製カッティングマシンのカッティングソフトウェアです。
図形や文字を入力してカットデータをデザインし、出力してシートをカットできます。


------------------------------------------------------

＜２. 必要なシステム構成＞

　・対応OS：Windows Vista(32ビット版、64ビット版), Windows 7 (32ビット版、64ビット版), Windows 8 (32ビット版、64ビット版), Windows 8.1 (32ビット版、64ビット版) のいずれかのオペレーティングシステム
　※本ソフトウェアは32ビットアプリケーションのため、64ビット版WindowsではWOW64（Windows-On-Windows 64）上で動作いたします。
・ Internet Explorer 4.0 以上
・ オペレーティングシステムが動作する CPU
・ オペレーティングシステムが動作する RAM
・ インストールに10Mbyte以上のハードディスク空き容量
・ 16bit色(High Color)以上をサポートするビデオカード


------------------------------------------------------

＜３. インストール＞

CutStudio は、インストールディスクから直接起動することはできません。
インストールディスクの内容をハードディスクに転送し、CutStudio を Windows で使用可能にします。
この作業を「インストール」といいます。


■CD-ROMでプログラムが供給されている場合
CD-ROMのセットアップメニューに従ってインストールしてください。
