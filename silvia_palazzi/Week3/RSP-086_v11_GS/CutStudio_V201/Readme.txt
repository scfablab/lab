1.  Summary 
========================================================

CutStudio is a cutting program for cutting machines from Roland DG Corp.
It enables you to enter shapes and text to design cutting data, then output the cutting data to cut media.


========================================================
2.  System requirements
========================================================

- Microsoft(R) Windows Vista (32-bit,64-bit), Windows 7 (32-bit,64-bit), Windows 8 (32-bit,64-bit) or  Windows 8.1 (32-bit,64-bit) operating system
- Internet Explorer 4.0 or higher
- The minimum required CPU for the operating system
- The minimum amount of required RAM for the operating system
- 10 Mbytes or more of free hard disk space for installation
- 16-bit color (High Color) or higher video card
