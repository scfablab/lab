＜１. 概要＞

Roland CutStudio Plugin for CorelDRAW は、Corel社のソフトウェア「CorelDRAW」のプラグインソフトウェアです。
CorelDRAWで作図したカット線のデータを Roland CutStudio へ出力します。


------------------------------------------------------

＜２. 対応するバージョン＞

このプラグインは、CorelDRAW X3/X4/X5/X6/X7 で動作します。


------------------------------------------------------

＜３. インストール＞

※Windows Vista/7/8/8.1 へインストールするときは、[管理者] または [Administrators] グループのメンバーとしてWindowsへログオンしてください。


■ CD-ROMでプログラムが供給されている場合

CD-ROMのセットアップメニューに従ってインストールしてください。


■ 弊社のホームページからダウンロードした場合

ダウンロードしたファイルを解凍すると、解凍先のフォルダに次のファイルがあります。

	CutStudioPlugIn.gms	プラグインソフトウェア
	CutStudioPlugIn.bmp	プラグインのアイコン画像
	Readmej.txt		このファイル

CutStudioPlugIn.gms と CutStudioPlugIn.bmp を次のフォルダにコピーしてください。

	＜CorelDRAW X3＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite 13]-[Draw]-[GMS]
	＜CorelDRAW X4＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X4]-[Draw]-[GMS]
	＜CorelDRAW X5＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X5]-[Draw]-[GMS]
	＜CorelDRAW X6＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X6]-[Draw]-[GMS]
	＜CorelDRAW X7＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X7]-[Draw]-[GMS]

＜４. インストール後にやること＞

このプラグインを使うには、インストール後に次の作業をします。

	・ツールバーへの起動ボタンの追加
	・起動ボタンのアイコン変更

下記の手順にしたがって作業を行ってください。


**********************************
　ツールバーに起動ボタンを追加
**********************************

1. CorelDRAWを起動し、新規ドキュメントを開きます。

2. [ツール]-[カスタマイズ] をクリックします。
   [オプション]画面が表示されます。

3. [カスタマイズ]をクリックし、サブメニューから[コマンド]をクリックします。

4. コマンドのプルダウンメニューから[マクロ]を選びます。
   マクロの内容が表示されます。

5. マクロのリストから[CutStudioPlugIn.ExecuteCutStudio.CutStudio]を選び、ツールバーにドラッグ＆ドロップします。
  既存のツールバー以外の場所にドラッグ＆ドロップすると、Roland CutStudio Plugin for CorelDRAW 用のツールバーが新規に作成されます。

6. [OK]をクリックし、オプション画面を閉じます。
   CorelDRAW を終了します。

次回から、CorelDRAWを起動すると、先ほど追加した位置に起動ボタンが表示されます。


*********************************
　起動ボタンのアイコンを変える
*********************************

1. 上記「ツールバーに起動ボタンを追加」の手順 1〜4 までを実行し、[オプション]画面の[マクロ]を選択した状態にします。

2. マクロのリストから[CutStudioPlugIn.ExecuteCutStudio.CutStudio]を選択し、[外観]タブをクリックします。

3. [インポート]をクリックします。

4. アイコンのリストから[ファイル]をクリックします。

5. 以下のフォルダ中のファイルから [CutStudioPlugIn.bmp] を選び、[開く]をクリックします。
	＜CorelDRAW X3＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite 13]-[Draw]-[GMS]
	＜CorelDRAW X4＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X4]-[Draw]-[GMS]
	＜CorelDRAW X5＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X5]-[Draw]-[GMS]
	＜CorelDRAW X6＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X6]-[Draw]-[GMS]
	＜CorelDRAW X7＞ [Program Files]-[Corel]-[CorelDRAW Graphics Suite X7]-[Draw]-[GMS]
  アイコンの絵が変わります。

6. [OK]をクリックし、[オプション]ダイアログを閉じます。
   起動ボタンのアイコンが指定のアイコンに変更されます。
